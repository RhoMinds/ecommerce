
  # Premium Car Accessories UI

  This is a code bundle for Premium Car Accessories UI. The original project is available at https://www.figma.com/design/aPz1GevCICKgLgIlHi7DFr/Premium-Car-Accessories-UI.

  ## Running the code

  Requires Node.js 18+.

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  Run `npm run build` to create a production build.

  Run `npm run preview` (or `npm start`) to preview the production build locally.
